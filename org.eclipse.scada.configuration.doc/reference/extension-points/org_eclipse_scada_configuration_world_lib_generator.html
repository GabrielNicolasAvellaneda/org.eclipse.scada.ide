Generator Processors

Identifier:
org.eclipse.scada.configuration.world.lib.generator

Since:
0.1.0

Description:
Defines some processors which take part in the generation process.

Configuration Markup:

<!ELEMENT extension (nodeElementProcessor | oscarProcessor)*>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT nodeElementProcessor (forClass)*>

<!ATTLIST nodeElementProcessor

forClass     CDATA #IMPLIED

factoryClass CDATA #REQUIRED>

A processor that processes elements directly under the Application class.


  • forClass - The class this processor works for. Either the attribute "forClass "or the child element "forClass" can be used. If both are specified all will be used.
  • factoryClass - The implementation class.

<!ELEMENT forClass EMPTY>

<!ATTLIST forClass

class CDATA #REQUIRED>

An alternative to the attribute "forClass" in order to list more classes that this processor can handle.


  • class - The actual class name.

<!ELEMENT oscarProcessor EMPTY>

<!ATTLIST oscarProcessor

class CDATA #REQUIRED>

A processor that will be called for all applications based on EquinoxApplication


  • class - The class implementing the processor
