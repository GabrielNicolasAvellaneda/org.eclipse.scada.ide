Driver Extension Point

Identifier:
org.eclipse.scada.configuration.infrastructure.lib.driver

Since:
0.1.0

Description:
An extension point for creating drivers in the world model.

Configuration Markup:

<!ELEMENT extension (driverFactory)*>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT driverFactory EMPTY>

<!ATTLIST driverFactory

typeId       IDREF #REQUIRED

factoryClass CDATA #REQUIRED>

Defines a driver factory that creates driver configuration instances for the "world" model.


  • typeId - The driver type id for which this factory creates instances.
  • factoryClass - The factory class that created drivers instances for the "world" configuration model.
